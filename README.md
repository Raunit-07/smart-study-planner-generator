# Smart Study Planner Generator

A simple React application that helps students create a study schedule based on their subjects, exam date, and study mode.

## Features

* Generate a day-by-day study plan
* Lazy and Hardcore study modes
* Motivational quotes
* Study summary
* Export study plan as PDF

## Technologies Used

* React.js
* JavaScript
* CSS
* jsPDF

## How to Run

1. Install dependencies


npm install


2. Start the development server


npm run dev


3. Open the URL shown in the terminal.
